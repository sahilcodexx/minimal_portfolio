## Overview

**That Startup** is a platform where entrepreneurs can submit their startup ideas and connect with like-minded founders.  
The platform is designed to encourage collaboration, idea sharing, and early-stage startup networking.

It is built using **Next.js**, **TypeScript**, **Tailwind CSS**, **Sanity**, and **NextAuth**.

---

## Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Sanity**
- **NextAuth**

---

## Features & Learnings

- Users can submit startup ideas using **Markdown (.md)** format.
- Integrated **Sanity CMS** for content management.
- Implemented authentication using **NextAuth**.
- Used **Next.js PPR (Partial Prerendering)** to combine static and dynamic rendering efficiently.
- Implemented **Sanity Write & Real-time APIs** to update and track idea view counts in real time.
- Fully responsive UI built with **Tailwind CSS**.
