### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sahilcodexx/minimal_portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd minimal_portfolio
   ```
3. **Launch the site:**
   Open `index.html` directly in your browser or use the "Live Server" extension in VS Code for real-time updates.

1. **Theming**: Open `assets/css/style.css` and update the `:root` variables (colors, fonts).
